    Pi-hole - Installation

    There are many different ways to install and run a pi-hole container.

    Automated - Fastest

    Run the command below to pull pi-hole container docker image and create a docker for pi-hole.

    Bash
    wget -qO- https://raw.githubusercontent.com/time0ut07/HomeServer/main/scripts/install_pi-hole.sh | bash
